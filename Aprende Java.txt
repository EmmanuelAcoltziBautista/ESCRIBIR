Para aprender a programar es necesario comprender la logica.

La logica en la programación es bastante simple solo es un
conjunto de instrucciones que se le indican a una maquina.

Y se basan en Condicionales, Bucles, Inputs y Ouputs.

Inputs:
Entrada de datos por medio del teclado.

Ouputs:
Salida de datos por la pantalla.

Condicionales:
Si se cumple entonces realiza la siguiente accion pero 
si no se cumple realiza la siguiente accion.

Bucles:
Los bucles se utilizan para repetir cierta accion y asi no volver
a reescribir las mismas lineas de codigo entre ellos 
se encuentran For, While y Do-While

	TIPOS DE DATOS

INT o Enteros:
Son aquellos numeros que no cuentan con
decimasles Ejemplo: 1, 2, 3...

Double o Real:
Son aquellos numeros que si cuentan con decimales
Ejemplo: 1.5, 2.5,...

String o Carater:
Son aquellos que puedes ingresar letras para poder y caracteres
especiales Ejemplo: hola, a, @$%...

Booleanos:
Los tipos booleanos son aquellos que se designan como true o false
cierto o falso su funcion es poder crear por ejemplo crear variables bandera
